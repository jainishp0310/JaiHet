# **App Name**: Heartbeat Countdown

## Core Features:

- Event Tracking: Track multiple romantic events with associated countdowns.
- Countdown Display: Display real-time countdowns to each specified date with personalized messages.
- Personalized AI Messages: Generate heartfelt, custom messages that trigger when a countdown ends. The tool will choose to reference either anniversary or birthday, depending on the nature of the event.
- Date Input and Validation: Allow users to input dates and validate the format, ensuring accurate countdowns.
- Event Tagging: Associate custom tags (e.g., 'Anniversary', 'Birthday') with each event for organization.

## Style Guidelines:

- Primary color: Soft coral (#F08080) for a warm, romantic feel.
- Background color: Pale pink (#FADADD), very lightly desaturated.
- Accent color: Dusty lavender (#D8BFD8), a sophisticated and contrasting hue.
- Body and headline font: 'Alegreya', a serif font known for its modern elegance, creates a sophisticated and readable interface, perfect for romantic messages.
- Use delicate, heart-themed icons to represent event types and navigation elements.
- Elegant, flowing layout with soft edges to enhance the romantic theme.
- Gentle fade-in and subtle transitions on countdown updates for a smooth user experience.
